# 微信小程序上线指南

恭喜你完成了数独小程序的开发！以下是上线发布到微信的完整流程。

## 1. 准备工作

### 1.1 确认 AppID

请确保 `miniprogram/project.config.json` 中的 `appid` 是你注册的正式小程序 ID。

- 当前配置: `"appid": "wxe3d8d8a5b70a4936"`
- 如果这是测试 ID，请在微信开发者工具右上角详情中点击“修改 AppID”进行替换。

### 1.2 真机预览

在上传代码前，务必在真机上进行最终测试：

1. 在微信开发者工具顶部点击 **预览**。
2. 使用手机微信扫描二维码。
3. 检查所有功能（特别是“退出拦截”和“计时器”）是否正常。

## 2. 上传代码

1. 在微信开发者工具右上角点击 **上传** 按钮。
2. 填写版本号（例如 `1.0.0`）和项目备注（例如“数独游戏发布版”）。
3. 点击上传，等待提示“上传成功”。

## 3. 提交审核

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)。
2. 进入 **管理** -> **版本管理**。
3. 在 **开发版本** 列表中找到刚才上传的版本。
4. 点击 **提交审核**。
5. 填写审核信息：
   - **功能页面**：选择首页 `pages/home/home`。
   - **标题**：数独大师。
   - **所在服务类目**：选择“游戏 > 休闲游戏”或其他适合类目（需确保已有相应资质）。
   - **标签**：数独, 益智, 休闲。
6. 提交后等待微信官方审核（通常 1-7 个工作日）。

## 4. 全量发布

1. 审核通过后，你会收到微信通知。
2. 再次登录微信公众平台 -> **版本管理**。
3. 在 **审核版本** 中找到已通过的版本。
4. 点击 **提交发布**。
5. 此时小程序已正式上线，用户可以通过搜索名称找到它！

## 5. 后续维护

- 如有 Bug 修复或新功能，重复上述“上传 -> 审核 -> 发布”流程即可。
- 记得在 `project.config.json` 或代码中更新版本号以便管理。
